## 📚 Detailed Week-by-Week Breakdown

### **Week 1: Core Algorithms (Days 1-7)**

| Day | Topic | Learning Outcomes | Key Concepts |
|-----|-------|------------------|---------------|
| 1-2 | Linear Regression | Predict continuous values | Cost function, Gradient descent, R² score |
| 3-4 | Logistic Regression | Binary classification | Sigmoid function, Decision boundary, Log loss |
| 5-6 | Decision Trees & RF | Multi-class classification | Information gain, Entropy, Ensemble learning |
| 7 | **Mini-Project** | Compare algorithms | Iris classification (all 3 algorithms) |

**Week 1 Deliverable**: Iris dataset with 3 algorithms compared + visualization report

---

### **Week 2: Data Processing & EDA (Days 8-14)**

| Day | Topic | Learning Outcomes | Key Concepts |
|-----|-------|------------------|---------------|
| 8-9 | Data Cleaning | Handle real messy data | Missing values, Outliers, Encoding |
| 10-11 | EDA & Visualization | Understand data deeply | Distributions, Correlations, Patterns |
| 12-13 | Feature Engineering | Create powerful features | Feature scaling, Selection, Creation |
| 14 | **End-to-End Project** | Full pipeline | Titanic survival prediction |

**Week 2 Deliverable**: Titanic dataset solution with EDA report + feature importance

---

### **Week 3: Model Evaluation & Tuning (Days 15-21)**

| Day | Topic | Learning Outcomes | Key Concepts |
|-----|-------|------------------|---------------|
| 15-16 | Metrics & Evaluation | Measure performance correctly | Accuracy, Precision, Recall, F1, AUC-ROC |
| 17-18 | Cross-Validation & Tuning | Optimize models | K-Fold, GridSearchCV, RandomizedSearchCV |
| 19-20 | Overfitting & Regularization | Improve generalization | Bias-Variance, L1/L2, Learning curves |
| 21 | **Capstone Project** | Apply all concepts | Boston Housing with full optimization |

**Week 3 Deliverable**: Boston Housing analysis with hyperparameter optimization report

---

### **Week 4: Ensemble Methods (Days 22-28)**

| Day | Topic | Learning Outcomes | Key Concepts |
|-----|-------|------------------|---------------|
| 22-23 | Gradient Boosting | Advanced ensemble methods | XGBoost, LightGBM, Feature importance |
| 24-25 | Advanced Ensembles | Combine multiple models | Stacking, Blending, Voting |
| 26-28 | **Kaggle Competition** | Real-world competition | End-to-end solution, Iteration, Ranking |

**Week 4 Deliverable**: Kaggle competition submission + documentation

---

### **Week 5: Consolidation (Days 29-35)**

| Day | Topic | Learning Outcomes | Key Concepts |
|-----|-------|------------------|---------------|
| 29-31 | Advanced Features | Deep feature engineering | Interactions, Polynomials, Domain knowledge |
| 32-33 | Dimensionality Reduction | Handle high-dimensional data | PCA, t-SNE, Feature importance |
| 34 | Anomaly Detection | Detect outliers/fraud | Isolation Forest, LOF |
| 35 | **Capstone Project** | Portfolio-ready project | Custom dataset, full documentation |

**Week 5 Deliverable**: Final capstone project with presentation-ready report

## 🛠️ Tools & Setup

### Required Libraries
```bash
pip install pandas numpy scikit-learn matplotlib seaborn jupyter \
            xgboost lightgbm pandas-profiling ipython
```

### Jupyter Setup
```bash
jupyter notebook
```

### Kaggle Setup
1. Create Kaggle account: https://www.kaggle.com/
2. Install Kaggle API: `pip install kaggle`
3. Download API token from Kaggle settings
4. Place in `~/.kaggle/kaggle.json`
